
 Lab 6e: Working with DBFS using Databricks CLI
 ----------------------------------------------
 
  NOTE: This is a continuation of Lab 4. 
  
  1. Make sure you have the Databricks CLI installed and configured (with a profile) on your local machine.
	    
  2. Open a Terminal and activate the virtual environment in which you have installed Databricks CLI
  
		cd E:\Databricks\Projects\dbcli-venv\Scripts
		activate
		
  3. Run the following commands to work with DBFS with a valid profile 
     (if 'default' is a valid profile for the workspace you can omit profile option in the commands)
	 
	 
		help
		---------------
		databricks -h
		databricks fs -h
		databricks fs mv -h
		
		ls - List files in DBFS
		-----------------------
		databricks fs ls     				(using 'default' profile)
		databricks fs ls --profile azure    (using 'azure' profile)
		
		databricks fs ls dbfs:/FileStore --profile azure
		databricks fs ls -l dbfs:/FileStore --profile azure
		databricks fs ls -l dbfs:/FileStore/tables --profile azure
		
		mkdirs - Make directories in DBFS.
		------------------------------------
		databricks fs mkdirs dbfs:/FileStore/tables/flights_json
		
		cp - Copy files to and from DBFS.
		----------------------------------
		databricks fs cp E:\Databricks\datasets\flight-data\2013-summary.json dbfs:/FileStore/tables/flights_json
		databricks fs cp --overwrite E:\Databricks\datasets\flight-data\2013-summary.json dbfs:/FileStore/tables/flights_json
		databricks fs cp -r E:\Databricks\datasets\flight-data\json dbfs:/FileStore/tables/flights_json
		
		mv - Moves a file between two DBFS paths.
		-------------------------------------------
		databricks fs mv dbfs:/FileStore/tables/flights_json dbfs:/FileStore/tables/flights_json2
				
		rm - Remove files from dbfs
		---------------------------
		databricks fs rm -r dbfs:/FileStore/tables/flights_json/csv
