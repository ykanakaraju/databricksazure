
  Lab 7 - Install and validate Azure CLI on Windows
  -------------------------------------------------- 
 
  1. Download and Install Azure CLI

	1.1 Download MSI installer from the following link
	    https://learn.microsoft.com/en-us/cli/azure/install-azure-cli-windows?tabs=azure-cli#install-or-update
	
	1.2 Double-click on the downloaded msi file and complete the installation


  2. Validate the Azure CLI installation

	2.1 Open the windows 'Power Shell' and run the following command
		az -v

	2.2 Login the Azure CLI using the following command
		az login
		
		NOTE:
		If the above does not work, try using the following command:
		
		az login --tenant TENANT_ID
		example: az login --tenant 5bd47fe1-d65b-4304-bda9-d9607cf1e2e5

	2.3 Enter the credentals in the browser window to login to your Azure account

	2.4 Back in the 'Power Shell' terminal, run the following commands to list resource groups
		az group list
		az group list --query [*].name

	2.5 Get help about resource group commands
		az group -h
		az group create -h
